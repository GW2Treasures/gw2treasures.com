# `@gw2treasures/tsconfig`

Common tsconfig files for gw2treasures projects.

## Useage

```json
{
  "extends": "@gw2treasures/tsconfig/base",
  "compilerOptions": {
    // ...
  },
}

```

## Installation

```
npm i -D @gw2treasures/tsconfig
```
